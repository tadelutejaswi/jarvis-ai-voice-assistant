# 🤖 JARVIS AI Voice Assistant

Your own movie-inspired voice-controlled AI assistant, built in Python with ChatGPT integration, voice recognition, and powerful features — all in one executable!

![Python](https://img.shields.io/badge/Python-3.12-blue.svg)
![License](https://img.shields.io/github/license/tadelutejaswi/jarvis-ai-voice-assistant)
![Stars](https://img.shields.io/github/stars/tadelutejaswi/jarvis-ai-voice-assistant?style=social)

---

### 🚀 Features

- 🎤 **Voice-activated commands**
- 🌦️ Real-time **Weather info**
- 📚 **Wikipedia** summaries
- 😂 Funny **Jokes**
- 🎬 **YouTube** video search
- 💬 Powered by **ChatGPT (OpenAI API)**
- 🧠 Memory/context awareness
- 🔊 Male voice + startup sound
- 🪟 Windows `.exe` file included!

---

### 📦 How to Use (Executable)

1. Go to the [Releases Page](https://github.com/tadelutejaswi/jarvis-ai-voice-assistant/releases)
2. Download `jarvis.exe`
3. Double-click & speak — JARVIS is ready to serve you.

---

### 🛠️ Tech Stack

- Python 3.12
- pyttsx3
- SpeechRecognition
- PyAudio
- OpenAI API
- Wikipedia API
- pyjokes
- dotenv
- pygame
